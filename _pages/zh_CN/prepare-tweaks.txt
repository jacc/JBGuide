## Tweaks

As you know from reading the home page, tweaks allow you to customize your device.

Some tweaks are not compatible with iOS 11. Please check the tweak compatibility list before installing any tweaks
{: .notice--warning}

## Installing Tweaks

To install tweaks, you will need a program that allows SFTP connections. For Windows, this would be `WinSCP`, and on macOS, `Cyberduck`

Once you download the program, find a tweak that you want to install. Usually, tweaks come in the form of `.deb` files. Using a program that allows unzipping of these files (`The Unarchiver` on macOS, and `Altap` on Windows). Once the file is unzipped, there should be three compressed files inside.
